# Frontend

This folder contains the browser-based website for NEXUS AI 2.0.

## Layout

- `index.html`
  The page shell and semantic markup.
- `assets/css/`
  Visual styling and responsive layout rules.
- `assets/js/app/`
  Page orchestration and state management.
- `assets/js/renderers/`
  DOM rendering utilities for chat, citations, and debug evidence.
- `assets/js/services/`
  API client code for mock and live backend modes.
- `assets/js/utils/`
  Shared browser utilities such as markdown rendering.
- `assets/js/fixtures/`
  Mock response payloads for frontend-only demos.
